My C++ template project.

# What is C++

https://en.cppreference.com/w/cpp/language/basic_concepts

# Setup

- clone
- `make tags`
    - tags `main.cpp` and all header dependencies

# Use

- Vim:
    - `Ctrl-X Ctrl-O` to omnicomplete using tags
        - e.g., insert `put` then `Ctrl-X Ctrl-O`
    - `:tselect <tag>`
        - e.g., `:tselect printf`
